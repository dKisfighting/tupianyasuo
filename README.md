# 图片压缩工具

这是一个简单易用的在线图片压缩工具，采用苹果风格设计，具有以下特点：

## 功能特性
- 支持上传 PNG、JPG 格式图片
- 实时预览原图和压缩后的效果
- 可调节压缩比例
- 显示压缩前后文件大小
- 支持下载压缩后的图片

## 技术实现
- 使用原生 HTML5、CSS3 和 JavaScript
- 使用 Canvas API 进行图片压缩
- 响应式设计，适配各种设备
- 采用 Flexbox 布局

## 项目结构
- index.html: 主页面结构
- styles.css: 样式表
- script.js: 交互逻辑
- upload-icon.svg: 上传图标
